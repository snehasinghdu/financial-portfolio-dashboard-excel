# Financial Portfolio Dashboard (Excel)

## Overview
This project is an interactive financial portfolio dashboard built using Microsoft Excel. It helps track investments, analyze portfolio performance, visualize asset allocation, and monitor returns efficiently.

## Features
- Portfolio performance tracking
- Investment allocation analysis
- Return calculations
- Interactive charts and visuals
- Financial insights dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Financial Formulas

## Project Files
- `Financial Portfolio Dashboard.xlsx`

## Dashboard Preview
- `Financial portfolio dashboard.png`

## Purpose
This project was created to strengthen practical finance and data visualization skills through a real-world portfolio tracking dashboard.
